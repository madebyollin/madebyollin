What is this?

Press WASD to explore a glitchy emulation of an old video game. Shift key (or double-click) for sprint.

How does it work?

A neural network was trained to predict the next video frame given the current frame + controls. The network is small and missing some critical information, so it's a bit glitchy, but oh well.
